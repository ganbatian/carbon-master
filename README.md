# CarbonMaster - 企业碳排放管理平台

企业级碳排放核算、报告生成、趋势分析平台，助力企业碳中和目标达成。

## 核心功能

- 📊 **数据概览** - 企业信息管理，碳排放数据一目了然
- 🔢 **碳核算引擎** - 支持 Scope 1/2/3 全范围碳核算
- 📈 **趋势分析** - 月度碳排放趋势图表展示
- 📋 **报告生成** - 一键生成合规碳排放报告
- ⚠️ **碳配额预警** - 超配额智能预警提醒

## 技术栈

- React 18 + Vite
- Tailwind CSS
- Recharts 图表库
- Lucide Icons

## 快速开始

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建生产版本
npm run build
```

## 碳核算范围

### Scope 1 - 直接排放
- 固定燃烧源（天然气、煤炭、柴油等）
- 移动燃烧源（公司车辆）
- 过程排放（如水泥生产）

### Scope 2 - 间接排放
- 外购电力

## 数据来源

- IPCC 第六次评估报告 (AR6)
- 中国生态环境部排放因子
- ISO 14064-1:2018
